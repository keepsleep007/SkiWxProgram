# SkiWxProgram
滑雪俱乐部微信小程序  
这是一个全自研滑雪俱乐部微信小程序，并且已经上线使用，已申请软著  
后端管理系统使用前后端分离架构，管理页面使用Vue3开发、后台服务使用java开发  
微信扫描此二维码可打开小程序体验
